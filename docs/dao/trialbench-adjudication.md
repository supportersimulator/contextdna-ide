Adjudicators evaluate blinded outputs against prespecified rubrics. They classify failures as:
- agent fault,
- missing context,
- ambiguous task,
- tool failure,
- valid exploration,
- architecture violation,
- governance paralysis.

Adjudicators must not infer or record arm labels during blinded review.
